## P3 - Exercise Execution Modes

Goal:
Support different exercise execution patterns.

Modes:

* Standard
* Unilateral
* Superset
* Circuit

Notes:
For unilateral exercises the rest timer should start after both sides are completed.

Examples:

* One-arm row
* Bulgarian split squat
* One-arm curls
* One-arm triceps extensions

Status:
Research needed before implementation.
